・作品３：タクシディ

実行ファイルや動画がサイズの問題で上げることができていないため、
もし、実際にプレイする場合やどんな作品か確認したい場合は以下の方法を行ってください。
・プレイしたい場合(タクシディを除く)
　・プレイしたい作品内の作品情報.txtを開く。
　・そのファイルの文末辺りに「実行ファイルアップロード先」という項目があると思います。
　・その横のURLをブラウザで検索していただくと、必要なデータがアップロードされているのでダウンロードしてください。
　・ダウンロードしたデータの中に「タイトル名.exe」がありますのでそれをダブルクリックするとプレイすることができます。
・動画が見たい場合(タクシディを除く)
　・作品情報から
　　・各作品の作品情報ファイルの文末にある「参考動画URL」の横にあるURLから視聴することができます。
　・ポートフォリオから
　　・ポートフォリオ内の各作品ごとのQRコードをクリックすることで視聴することができます。
